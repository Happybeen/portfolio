Skill

웹 퍼블리셔가 되고자 하는 저의 능력입니다

HTML

웹 표준과 웹 접근성을 준수하는 HTML 정보 구조설계로 웹사이트 퍼블리싱을 해오고 있습니다. 웹사이트를 퍼블리싱 하기 전에 미리 사이트 사용 목적, 클래스 네임 구성, HTML 와이어프레임까지 미리 정리해 놓고 체계적인 퍼블리싱 작업을 수행합니다.

CSS

퍼블리싱 작업을 하면서 크로스브라우징(Cross Browsing)의 중요성을 생각하면서 안정적인 CSS 속성을 하용하며 HTML 구성요소를 배치하고 디자인을 합니다. SASS(SCSS)를 사용하여 간단한 예제를 만들 수 있습니다.

Javascript

탭 메뉴, 네비게이션 등 웹 퍼블리싱을 하는데 꼭 필요한 제이쿼리를 활용합니다. if 조건문, for 반복문 등 필수적인 자바스크립트 활용이 가능합니다. 주로 제이쿼리와 병행해서 사용합니다.

Responsive
Web

반응형 웹 제작에 있어서 기존 방식인 Float, Position, Media Query를 활용한 반응형 웹 제작 경험을 가지고 있습니다. 또한, 대부분 기존 방식으로 작업하지만, Flex와 Grid 방식을 활용하여 반응형 웹 사이트 제작도 가능합니다.

Design Program

웹사이트를 구성하는 배너 디자인 및 웹 UI 디자인 작업을 위해 포토샵과 일러스트레이터 작업을 해본 적이 있습니다. 피그마의 경우 프로토타입을 제작한 경우가 있으며, 웹과 모바일 버전의 디자인을 만들어 경험이 있습니다.

Text Editor

퍼블리싱을 하면서 주로 사용한 텍스트에디터는 Visual Studio Code입니다. 필요에 따라 Adobe Bracket를 병행해서 사용합니다.
